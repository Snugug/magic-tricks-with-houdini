We'll throw a button in as the first element that only toggles the `data-state` attribute on and off on its parent. Paired with registered properties, everything else is good old fashioned CSS and transitions. No external dependencies, works with any framework, infinitely distributable. All the makings of an ideal pattern for a design system. ---

Animated Circular Navigation Menu

.circle { --item-size: 32; --padding: 32; --angle: 0; display: layout(circle); transition-property: --angle, --padding, fill; } .circle[data-state="active"] { --padding: 32; --angle: 0; }
