There are a few very draft-quality specs that are part of Houdini as well, but there's not enough in them to really cover, yet

  • CSS Parser API - Parse CSS like string (individual rules and whole rulesets) in to Typed OM
  • Font Metrics API Level 1 - Provide basic fonts metrics, like width of font line box, height between the highest top and lowest bottom of em squares, and baselines of all given text
  • Box Tree API Level 1 - An API for accessing information about how boxes are displayed and positioned from a layout
